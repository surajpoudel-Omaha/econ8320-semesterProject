ECON 8320 Semester Project

Labor Market Data Analysis Dashboard

Overview

This project was developed for an ECON 8320 course and focuses on analyzing U.S. labor market data using Python. It provides an interactive dashboard to explore trends in unemployment, nonfarm workers, and hours worked.

The application allows users to analyze economic patterns through dynamic visualizations and filtering.

Features
Interactive dashboard built with Streamlit
Visualizations using Plotly
Data processing and transformation with Pandas
Filtering by year and month
Simple interface

Tech Stack
  Python
  Pandas
  Plotly
  Streamlit
